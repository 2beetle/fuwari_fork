# AI总结功能使用指南

## 功能概述

AI总结功能为您的博客文章提供智能生成的摘要，帮助读者快速了解文章核心内容。

## 功能特点

✅ **多语言支持** - 支持中文、英文、日文、西班牙文
✅ **响应式设计** - 完美适配桌面端和移动端
✅ **主题适配** - 支持深色和浅色主题
✅ **可折叠界面** - 用户可以展开/收起AI总结
✅ **优雅动画** - 平滑的过渡效果和交互体验

## 如何使用

### 1. 为文章添加AI总结

在文章的 FrontMatter 中添加以下字段：

```yaml
---
title: 文章标题
published: 2025-01-01
# ... 其他字段

# AI Summary
aiSummary: "这里填写AI生成的总结内容"
aiSummaryGenerated: 2025-11-26  # 生成日期
aiSummaryModel: "GLM-4.6"  # 使用的AI模型
---
```

### 2. 字段说明

- `aiSummary` - AI总结的内容（必需）
- `aiSummaryGenerated` - AI总结的生成日期（可选）
- `aiSummaryModel` - 使用的AI模型名称（可选）

### 3. 显示效果

- 当文章包含 `aiSummary` 字段时，AI总结模块会自动显示
- 总结模块位于文章内容和许可证信息之间
- 用户可以点击标题栏展开/收起总结内容
- 显示生成时间和AI模型信息

## 技术实现

### 组件结构
```
src/components/AISummary.astro  # AI总结显示组件
src/content/config.ts           # 扩展了文章Schema
src/i18n/                      # 添加了多语言支持
src/pages/posts/[...slug].astro # 集成到文章页面
```

### 样式设计
- 使用CSS变量实现主题适配
- 渐变色边框突出AI功能
- 优雅的hover效果和过渡动画
- 移动端优化布局

### 多语言支持
支持以下语言的界面文本：
- 🇨🇳 中文：AI 总结
- 🇺🇸 英文：AI Summary
- 🇯🇵 日文：AI まとめ
- 🇪🇸 西班牙文：Resumen IA

## 自定义配置

### 修改样式
您可以编辑 `src/components/AISummary.astro` 中的CSS来自定义样式：

```css
.ai-summary-container {
  /* 自定义容器样式 */
}

.ai-icon-container {
  /* 自定义图标样式 */
}
```

### 添加新语言
1. 在 `src/i18n/i18nKey.ts` 中添加新的翻译键
2. 在 `src/i18n/languages/` 目录下为新语言添加翻译文件
3. 更新组件中的国际化调用

### 调整显示位置
目前AI总结显示在文章内容之后，许可证之前。如需调整位置，请编辑 `src/pages/posts/[...slug].astro` 文件中的组件顺序。

## 最佳实践

1. **总结内容** - 保持简洁明了，突出文章核心观点
2. **长度控制** - 建议总结长度在100-200字之间
3. **更新时机** - 当文章内容有重大更新时，同步更新AI总结
4. **质量检查** - 确保AI总结准确反映文章内容

## 故障排除

### AI总结不显示
- 检查 FrontMatter 中是否包含 `aiSummary` 字段
- 确认字段值不为空
- 查看浏览器控制台是否有错误信息

### 样式问题
- 检查CSS变量是否正确定义
- 确认主题切换功能正常工作
- 验证响应式断点设置

### 多语言问题
- 检查语言文件是否完整
- 确认翻译键值匹配
- 验证语言切换逻辑
